WordPress custom scrollbar is highly customizable WordPress plugin. 

== Description ==

WordPress custom scrollbar is highly customizable WordPress plugin. Features include vertical and/or horizontal scrollbar(s), adjustable scrolling momentum, mouse-wheel, keyboard and touch support, ready-to-use themes and customization via CSS, RTL direction and many more.

You can change almost 100+ demo style, ScrollBar scroll speed, Showing ScrollBar arrow button mode, ScrollBar auto hide mode, ScrollBar auto expand mode, ScrollBar moving mode by keyboard & other settings by using the options and shortcode of this plugin.
[Click here](http://wordpress-custom-scrollbar.softhopper.net/) to see a live demo

**Features include:**

* 100+ scrollbar demo
* Full page Scrollbar
* Single Content Scrollbar
* Custom Selector Scrollbar
* Vertical and horizontal scrollbar support
* Highly Customizable
* Enable TinyMCE Button
* ScrollBar scroll speed control
* ScrollBar arrow button mode control
* ScrollBar auto hide mode
* ScrollBar auto expand mode
* ScrollBar moving mode by keyboard
* Shortcode Powered
* Translation Ready
* Responsive for all devices
* All major browser supported
* Easy to handle

== Installation ==

= Installation =

1. Upload the `custom-content-scrollbar` folder to the `/wp-content/plugins/` directory
1. Activate the Custom Content Scrollbar plugin through the 'Plugins' menu in WordPress
1. After activing the plugin will add a menu under Settings called "SH Scrollbar Options".
