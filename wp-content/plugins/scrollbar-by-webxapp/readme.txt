= WordPress custom scrollbar =

WordPress custom scrollbar is highly customizable WordPress plugin. Features include vertical and/or horizontal scrollbar(s), adjustable scrolling momentum, mouse-wheel, keyboard and touch support, ready-to-use themes and customization via CSS.
